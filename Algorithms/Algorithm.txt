Top Algorithm Used in Machine Lerning

Linear Regression
Logistic Regression
Decision Trees
Random Forests
Support Vector Machines (SVM)
K-Nearest Neighbors (KNN)
Naive Bayes
K-Means Clustering
Principal Component Analysis (PCA)
